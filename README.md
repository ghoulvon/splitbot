# 🤖 Ортақ Есеп — бот для общих расходов

## Возможности
- ➕ Добавить расход и разделить на 2 или 3
- 💰 Смотреть кто кому сколько должен
- 💸 Отметить перевод и получить подтверждение
- 📋 История расходов

## Команды
| Команда | Описание |
|---------|----------|
| `/start` | Регистрация в боте |
| `/add` | Добавить расход |
| `/balance` | Текущий баланс |
| `/pay` | Отметить что перевёл деньги |
| `/history` | Последние 10 расходов |

## Установка на Railway (бесплатно)

### Шаг 1 — Получить токен
1. Открой Telegram → найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя и username боту
4. Скопируй токен вида `7123456789:AAF...`

### Шаг 2 — Загрузить код на GitHub
1. Зайди на [github.com](https://github.com) → создай аккаунт
2. Нажми **New repository** → назови `splitbot`
3. Загрузи все файлы (bot.py, requirements.txt, Procfile)

### Шаг 3 — Запустить на Railway
1. Зайди на [railway.app](https://railway.app) → войди через GitHub
2. Нажми **New Project** → **Deploy from GitHub repo**
3. Выбери свой репозиторий `splitbot`
4. Перейди в **Variables** → добавь:
   - Key: `BOT_TOKEN`
   - Value: твой токен от BotFather
5. Нажми **Deploy** — готово!

### Шаг 4 — Пригласить участников
Скинь всем ссылку на бота (например `t.me/ordak_esep_bot`)
Каждый должен написать `/start` чтобы зарегистрироваться

## Как пользоваться

**Добавить расход:**
```
/add → введи сумму → описание → выбери ÷2 или ÷3
```

**Посмотреть баланс:**
```
/balance
```

**Закрыть долг:**
```
/pay → выбери кому перевёл → получатель подтверждает
```
